ℹ️ Como atualizar o dashboard

Siga os passos:
  1. Selecione a planilha .xlsx exportada do sistema
  2. O admin processa os dados direto no navegador (sem servidor)
  3. Visualize o preview dos dados antes de confirmar
  4. Clique em "Baixar data.js" e substitua o arquivo na pasta do site
  5. O index.html atualiza automaticamente
