Como atualizar o dashboard

Siga os passos abaixo para atualizar os dados:
  1. Selecione a planilha (.xlsx) exportada do sistema de atendimentos
  2. O admin irá processar automaticamente os dados por fornecedor
  3. Visualize o preview antes de confirmar
  4. Clique em "Baixar data.js" e substitua o arquivo na pasta do site
  5. O dashboard (index.html) será atualizado automaticamente
